# ch04 조건문과 반복문

## 코드 실행 흐름 제어

> 코드 실행 흐름: 프로그램이 위에서 아래로만 실행되지 않고, 조건이나 반복에 따라 ‘건너뛰거나, 되돌아가거나, 멈추는 것’
> 
- 프로그램이 어디로 가느냐를 정하는 규칙을 말한다.
- 개발자가 원하는 방향으로 바꿀 수 있도록 해주는 것이 흐름 제어문이다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image.png)

제어문은 조건식과 중괄호 { } 블록으로 구성한다.

조건식의 연산 결과에 따라 블록 내부의 실행 여부가 결정된다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%201.png)

### 조건문(Conditional Statements)

→ 특정 조건이 참(true)일 때만 코드를 실행한다.

루핑: 조건문일 경우는 정상 흐름으로 돌아오지만 반복문일 경우    는 제어문 처음으로 다시 되돌아가 반복 실행

- if, else if, else : 조건에 따라 다른 블록을 실행
    - 만약 ~하면 이거, 아니면 저거
- switch case : 여러 값 중 하나와 일치하는 경우를 선택하여 실행
- 제어문 블록 내부에는 또 다른 제어문이 사용 가능하다.
    - if 문 내부에는 for 문이나 while 문도 가질 수 있기 때문에 개발자가 원하는 매우 복잡한 흐름 제어도 가능하다.

### 반복문(Loop Statements)

→ 특정 코드 블록을 여러 번 반복해서 실행한다.

- for : 정해진 횟수만큼 반복
- while: 특정 조건이 참인 동안 계속 반복
- do-while: 일단 한 번 실행 후 조건을 검사하여 반복

### 분기문(Branching Statements)

→ 반복문이나 함수 내부의 실행 흐름을 즉시 변경한다.

- break: 현재 실행 중인 반복문이나 switch 문을 즉시 종료하고 빠져나옴.
- continue: 현재 반복을 건너뛰고 다음 반복을 즉시 시작.
- return: 함수 실행을 종료하고 호출한 곳으로 돌아간다.

### 필요한 이유?

- 일단 없으면 굉장히 비효율적이다.
    
    → 필요 없는 코드도 전부 실행해야 한다.
    
- 오류가 많이 생긴다.
    
    → 예외 상황 처리가 불가
    
- 현실 부적합
    
    → 사용자 입력, 조건 반영 불가
    

다시 말하자면 사용자의 논리 구현과 작업 자동화 그리고 프로그램 제어 때문에 필요하다.

## if 문

 

> 프로그래밍에서 조건에 따라 코드의 실행 흐름을 제어하는 조건문으로, ‘만약 ~라면’이라는 의미를 가진다.
> 

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%202.png)

- 특정 조건이 참(true)일 때만 지정된 명령을 실행하거나, 거짓(false)일 경우 실행하지 않거나 다른 처리를 하도록 하는 문법이다.
- 중괄호 { } 블록 내에 실행문이 하나밖에 없다면 생략이 가능하다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%203.png)

생략이 가능하지만 하지 않고 작성하는 것이 좋다. 

- 코드의 가독성이 좋지 않다.
- 버그 원인이 될 수 있다.

```java
package statements;

public class IfExample {
    public static void main(String[] args) {

        int score = 93;
        if (score >= 90)
        {
            System.out.println("점수가 90보다 같거나 높습니다.");
            System.out.println("등급은 A입니다.");
        }
        if (score < 90)
            System.out.println("점수가 90보다 작습니다.");
            System.out.println("등급은 B입니다.");
    }
}

```

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%204.png)

위 코드는 개발자의 의도랑 다르다. if문은 true일 때만 실행해야 하는데 지금 { } 가 없어서 실행 결과가 이상한 것을 볼 수 있다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%205.png)

### if else 문 블록

if 문은 else 블록과 함께 사용되어 조건식의 결과에 따라 실행 블록을 선택할 수 있다.

- true이면 if 문 블록이 실행한다.
- false이면 else 블록이 실행한다.

```java
package statements;

public class IfElseExample {
    public static void main(String[] args) {

        int score = 85;

        if (score >= 90)
        {
            System.out.println("점수가 90보다 크거나 같습니다.");
            System.out.println("등급은 A입니다.");
        }
        else
        {
            System.out.println("점수가 90보다 작습니다.");
            System.out.println("등급은 B입니다.");
        }

    }
}

```

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%206.png)

### 조건문이 여러 개인 if 문

→ if 와 else 사이에 else if가 들어간다.

- else if가 true이면 해당 블록이 실행된다.
- else if의 수는 제한이 없다.
- 여러 개의 조건식 중 true가 되는 else if 블록만 실행하고 전체 if 문을 벗어나게 된다.
- 마지막에는 else 블록을 추가할 수 있는데, 모든 조건식이 false 일 경우 else 블록을 실행하고 if 문을 벗어나게 된다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%207.png)

```java
package statements;

public class IfElseIfElseExample {
    public static void main(String[] args) {

        int score = 75;

        if (score >= 90)
        {
            System.out.println("점수가 100~90입니다.");
            System.out.println("등급은 A입니다.");
        }
        else if (score >= 80)
        {
            System.out.println("점수가 80~89입니다.");
            System.out.println("등급은 B입니다.");

        }
        else if (score >= 70)
        {
            System.out.println("점수가 70~79입니다.");
            System.out.println("등급은 C입니다.");
        }
        else
        {
            System.out.println("점수가 70 미만입니다.");
            System.out.println("등급은 D입니다.");
        }
    }
}

```

### 주사위 예제

주사위를 굴려서 나올 수 있는 1, 2, 3, 4, 5, 6 중에서 하나의 수를 출력하는 코드를 작성해 보았다. 임의의 정수를 뽑기 위해 Math.random() 메소드를 활용해서 작성해 보았다.

```java
package statements;

public class IfDiceExample {
    public static void main(String[] args) {

        int num = (int) (Math.random() * 6) + 1;

        if (num == 1)
        {
            System.out.println("1번이 나왔습니다.");
        }
        else if (num == 2)
        {
            System.out.println("2번이 나왔습니다.");
        }
        else if (num == 3)
        {
            System.out.println("3번이 나왔습니다.");
        }
        else if (num == 4)
        {
            System.out.println("4번이 나왔습니다.");
        }
        else if (num == 5)
        {
            System.out.println("5번이 나왔습니다.");
        }
        else
        {
            System.out.println("6번이 나왓습니다.");
        }

    }
}

```

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%209.png)

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2010.png)

랜덤이라서 실행할 때마다 나오는 번호가 다르다.

```java
0.0  <=  Math.random()  <  1.0
```

위 메소드는 0.0 ≤ ~ 1.0 사이의 double 타입 난수를 리턴한다.

```java
(0.0 * 6)  <=  (Math.random() * 6)  <  (1.0 * 6)
```

여기에서 각 변에 6을 곱해서 0.0 ≤ ~ < 6.0 사이의 double 타입 난수를 얻게 된다.

```java
(int) 0.0  <=  (int) (Math.random() * 6)  <  (int) 6.0
```

각 변을 이제 int 타입으로 강제 타입 변환하면 0 ~ 5 중에서 하나의 정수 난수를 얻게 된다.

```java
(0+1)  <=  ((int) (Math.random() * 6) + 1)  <  (6+1)
```

마지막으로 각 변에 1을 더하면 이제 1 ~ 6 중에서 하나의 정수 난수를 얻게 된다.

```java
int num = (int) (Math.random() * n) + start;
```

그럼 이제 start부터 시작하는 n개의 정수 중에서 하나의 정수를 얻기 위한 코드를 작성 할 수 있다.

### 중첩 if 문

- if 문의 블록 내부에는 또 다른 if 문을 사용할 수 있다.
    
    → 이것을 중첩 if 문이라 부르는데, 중첩의 단계는 제한이 없다.
    

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2011.png)

한 번만 중첩되어도 매우 복잡한 실행 흐름이 생기는데 실제 프로그램에서도 여러 단계로 중첩 되는 경우가 많아서 잘 사용해야 한다.

```java
package statements;

public class IfNestedExample {
    public static void main(String[] args) {

        int score = (int) (Math.random() * 20) + 81;

        System.out.println("점수: " + score);

        String grade;

        if (score >= 90)
        {
            if (score >= 95)
            {
                grade = "A+";
            }
            else
            {
                grade = "A";
            }
        }
        else
        {
            if (score >= 85)
            {
                grade = "B+";
            }
            else
            {
                grade = "B";
            }
        }

    }
}

```

## switch 문

> 특정 변수나 표현식의 값에 따라 여러 경우(case) 중 하나를 선택하여 코드를 실행하는 제어문을 말한다.
> 
- if 문은 코드가 많이 복잡해 지지만 switch 문은 변수의 값에 따라서 실행문이 결정되기 때문에 조금 더 간결해 보인다.
- 괄호 안의 변수값에 따라 해당 case로 가서 실행문을 실행시킨다.
- 변수값과 동일한 값을 갖는 case가 없으면 default로 가서 실행문을 실행시킨다.
- default가 필요 없다면 생략 가능하다.
- 괄호에는 정수 타입과 문자열 타입 변수를 사용할 수 있다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2014.png)

```java
package statements;

public class SwitchExample {
    public static void main(String[] args) {

        int num = (int) (Math.random() * 6) + 1;

        switch (num)
        {
            case 1:
                System.out.println("1번이 나왔습니다.");
                break;

            case 2:
                System.out.println("2번이 나왔습니다.");
                break;
                
            case 3:
                System.out.println("3번이 나왔습니다.");
                break;

            case 4:
                System.out.println("4번이 나왔습니다.");
                break;

            case 5:
                System.out.println("5번이 나왔습니다.");
                break;

            default:
                System.out.println("6번이 나왔습니다.");

        }
    }
}

```

### break

- 다음 case를 실행하지 않고 switch 문을 빠져나가기 위해 필요하다.
- break가 없다면 다음 case가 연달아 실행되는데, 이때는 case 값과는 상관없이 실행된다.

---

- Java 12 이후부터는 Expressions(표현식)을 사용할 수 있는데 break 문을 없애는 대신에 화살표와 중괄호를 사용해 가독성이 좋아졌다.

```java
package statements;

public class SwitchExpressionExample {
    public static void main(String[] args) {

        char grade = 'B';
        
        switch (grade) 
        {
            case 'A', 'a' -> 
            {
                System.out.println("우수 회원입니다.");
            }
            case 'B', 'b' -> 
            {
                System.out.println("일반 회원입니다.");
            }
            default -> 
            {
                System.out.println("손님입니다.");
            }
        }
        switch (grade) 
        {
            case 'A', 'a' -> System.out.println("우수 회원입니다.");
            case 'B', 'b' -> System.out.println("일반 회원입니다.");
            default -> System.out.println("손님입니다.");
        }
    }
}

```

### Switch Expressions

→ 사용하면 스위치 된 값을 변수에 바로 대입할 수도 있다.

- 단일 값일 경우에는 화살표 오른쪽에 값을 기술한다.
- 중괄호를 사용할 경우에는 yield(Java 13부터 사용 가능) 키워드로 값을 지정하면 된다. 단 이 경우에는 default가 반드시 존재해야 한다.

 

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2018.png)

```java
package statements;

public class SwitchValueExample {
    public static void main(String[] args) {

        String grade = "B";

        int score1 = 0;

        switch (grade)
        {
            case "A":
                score1 = 100;
                break;

            case "B":
                int result = 100 - 20;
                score1 = result;
                break;

            default:
                score1 = 60;
        }

        System.out.println("score1: " + score1);

        int score2 = switch (grade)
        {
            case "A" -> 100;

            case "B" ->
            {
                int result = 100 - 20;
                yield result;
            }
            default -> 60;
        };

        System.out.println("score2: " + score2);

    }
}

```

## for 문

> 프로그램에서 반복해야 할 코드를 정해진 횟수만큼 또는 특정 조건이 만족될 때까지 반복 실행하는 제어문을 말한다.
> 
- 반복할 횟수를 미리 알고 있을 때 주로 사용한다.
- 대부분 초기화 → 조건식 → 증감문 순으로 구성한다.
    
    → 조건식이 true이면 실행문을 실행시키고 false이면 for 문을 종료하고 블록을 건너뛴다. 
    

```java
package statements;

public class PrintFrom1To10Example {
    public static void main(String[] args) {

        for(int i=1; i<=10; i++)
        {
            System.out.print(i + " ");
        }
    }
}

```

### 초기화식

- 조건식과 실행문, 증감식에서 사용할 변수를 초기화하는 역할을 한다.
- 초기화식이 둘 이상이 있을 수도 있고, 증감식도 둘 이상 있을 수 있다.
    
    → 이런 경우에는 쉼표(,)로 구분해서 작성한다.
    
    ```java
    for (int i=0, j=100;  i<=50 && j>=50;  i++, j--) { ... }
    ```
    
- 초기화식에서 선언된 변수는 for 문 블록 안에서만 사용되는 로컬 변수이다.
    - for 문을 벗어나서도 사용하고 싶으면 for 문 이전에 선언해야 한다.
    
    ```java
    int i;                              //for 문 이전에 변수 i 선언
    for (i=1; i<=100; i++) { ... }      
    System.out.println("종 i값: " + i); //for 문을 벗어나서 i를 사용
    ```

### 중첩된 for 문

> for 문 안에 또 다른 for 문이 들어간 구조를 말한다.
> 
- 바깥 for 문이 한 번 실행할 때마다 안 쪽 for 문이 지정 횟수만큼 반복한 다음에 다시 바깥 for 문이 반복한다는 것이다.

```java
package statements;

public class MultiplicationTableExample {
    public static void main(String[] args) {

        for (int m = 2; m <= 9; m++)
        {
            System.out.println("*** " + m + "단 ***");

            for (int n = 1; n <= 9; n++)
            {
                System.out.println(m + " x " + n + " = " + (m * n));
            }
        }
    }

}

```

## while 문

> 조건이 참(true)인 동안 계속 반복하는 구조를 말한다.
> 
- 조건식이 true일 경우에 계속해서 반복하고, false가 되면 반복을 멈추고 while 문을 종료한다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2025.png)

- 처음 실행될 때 조건식을 평가해서 true이면 실행하고 다시 반복하다가 false라면 종료한다.

```java
package statements;

public class PrintFromExample {
    public static void main(String[] args) {

        int i = 1;

        while (i <= 10)
        {
            System.out.print(i + " ");
            i++;
        }

    }
}

```

## break 문

> 반복문이나 조건문인 switch 문을 즉시 종료시키는 제어문이다.
> 

→ 조건이 만족되면, 남은 반복을 전부 건너뛰고 탈출한다는 점이다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2035.png)

```java
package statements;

public class BreakExample {
    public static void main(String[] args) {

        Outter: for (char upper = 'A'; upper <= 'Z'; upper++)
        {

            for (char lower = 'a'; lower <= 'z'; lower++)
            {
                System.out.println(upper + "-" + lower);

                if (lower == 'g')
                {
                    break Outter;
                }
            }
        }
    }
}
```

a ~ z까지 반복문을 작성했지만 g가 되면 break 하도록 설정해서 g가 나와서 종료되었다.

## continue 문

> 반복문에서 현재 반복만 건너뛰고 다음 반복으로 바로 넘어가는 제어문이다.
> 
- 반복문인 for 문, while 문, do-while 문에서만 사용되는데 대개 if 문과 같이 사용된다.
- 반복문은 유지하지만 특정 조건을 만족하는 경우에는 다음 반복으로 넘어간다.

![image.png](ch04%20%EC%A1%B0%EA%B1%B4%EB%AC%B8%EA%B3%BC%20%EB%B0%98%EB%B3%B5%EB%AC%B8/image%2037.png)

```java
package statements;

public class ContinueExample {
    public static void main(String[] args) {

        for(int i = 1; i <= 10; i++) 
        {
            if (i % 2 != 0) 
            {
                continue;
            }
            
            System.out.print(i + " ");
        }
    }
}

```

1 ~ 10 까지 반복하도록 되어 있지만 continue 문을 이용해서 짝수만 출력하고 홀수는 다음 반복으로 넘어간다.
